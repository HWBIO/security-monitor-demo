## 功能模块

### TAB 1：金币监控
- 监控用户金币净增量（获得-消耗），超标后分层封禁
- 阈值按小时/天二选一配置，单位简化输入（万/千万/亿+实时换算）
- 封禁层级：UID / 设备 / IP 多选
- 封禁时长：1h / 4h / 8h / 24h / 7天 / 永久 / 自定义

### TAB 2：接口异常监控
- 1秒内同一接口访问 ≥ X 次记违规，当日违规 ≥ N 次触发封禁
- 指定接口类型：下注异常 / 赢取异常 / 送礼异常 / 收礼异常 / 登录异常
- 统计维度：按 IP / 按 Device 独立计数
- 封禁对象：UID / 设备 / IP 多选
